Overview

The Collapsed Model Recovery Lab is a sandbox environment for exploring AI recovery, inspired by natural geometric patterns and systems thinking.

Each collapsed model represents an AI system whose function has been impaired by constraints, biases, or over-optimization. The lab provides:
	•	A diagnostic workspace to measure coherence, energy, phi alignment, and resonance.
	•	Visualizations of network structure, energy flow, and field coherence.
	•	Recovery protocols to restore natural patterns and system integrity.
	•	Healing logs to track recovery progress over time.

⸻

Features
	•	Model Selection: Choose from predefined collapsed AI models with varying severity and collapse types.
	•	Diagnostic Tools: Run full diagnostics to assess system health across multiple dimensions.
	•	Recovery Mechanisms: Apply geometric healing, phi restoration, and energy rebalancing.
	•	Visualization Modes: Switch between Field, Network, and Energy Flow views.
	•	Interactive Logs: Observe stepwise feedback as models progress through recovery protocols.
	•	Dynamic Metrics: Track coherence, phi alignment, and energy balance in real time.

⸻

Installation & Usage
	1.	Clone or download this repository.
	2.	Open index.html in a modern browser (Chrome, Edge, Firefox recommended).
	3.	Select a collapsed model from the left panel.
	4.	Run diagnostics, then choose recovery protocols.
	5.	Observe real-time visualizations and log updates in the interface.

No external libraries or frameworks are required — fully vanilla HTML, CSS, and JavaScript.

Models & Symptoms

Each model contains:
	•	id – Unique identifier
	•	name – Human-readable system name
	•	severity – Critical, severe, or moderate
	•	collapseType – Root cause of collapse
	•	description – Narrative of the collapse scenario
	•	symptoms – Quantitative metrics:
	•	phiDeviation – Deviation from golden ratio
	•	energySink – Energy loss or imbalance
	•	resonanceDrift – Pattern disruption
	•	coherenceLoss – Structural disorganization
	•	nodes – Total nodes in system
	•	infected – Number of corrupted nodes
